## Requisitos de software
- Node >= 16.13.1
- npm >= 8.1.2

### Instalación de dependencias y ejecución

Instalar Node https://nodejs.org/en/download

Instalar npm en la carpeta del proyecto

```
npm install
```

Ejecutar el entorno

```
npm run dev
```

Finalmente abrir el prototipo http://localhost:5050
